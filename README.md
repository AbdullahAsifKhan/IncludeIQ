# IncludeIQ — AI Chief Accessibility Officer

IncludeIQ is an AI-powered accessibility intelligence platform that helps organizations identify workplace accessibility barriers before they impact employees.

Using Microsoft Work IQ, Foundry IQ, and Fabric IQ, IncludeIQ analyzes workplace content such as job descriptions, policies, onboarding materials, and meeting transcripts to detect inclusion risks, uncover organizational patterns, and generate evidence-backed recommendations.

### Key Features

* Accessibility & Inclusion Risk Scoring
* Organization-wide Pattern Detection
* Evidence-backed Recommendations
* Executive Accessibility Reports
* Accessibility Roadmap Generation

### Why IncludeIQ?

Most accessibility tools focus on websites.

IncludeIQ focuses on organizations.

Instead of asking **"Is this webpage accessible?"**, IncludeIQ helps answer **"Is this workplace accessible?"**

### Architecture

Work IQ → Workplace Understanding

Foundry IQ → Accessibility Knowledge & Evidence

Fabric IQ → Organizational Reasoning & Pattern Detection
